# shadcn-template-react

A react js project template with shadcn ui library installed.

Kiển trúc của một project:
page chứa logic toàn bộ của hook. bao gồm việc load toàn bộ component, toàn bộ đối tượng
các component con sẽ chứa riêng từng component cá nhân, logic update hay sửa xoá đối tượng sẽ được truyền qua các function và các đối tượng
